## 如何启动dev
1. 安装node
2. 使用npm包管理工具，推荐使用<a href="https://www.npmjs.com/package/tyarn" target="_blank">tyarn</a>
3. 安装依赖：  `$npm i` or `$tyarn`
4. 启动webpack server： `npm run dev` or `tyarn dev`
